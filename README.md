Frontend Evaluacion Mundo Pacifico

Requisitos:

* npm version 9.8.0
* node version 20.5.1

Instrucciones:
1) Clonar repositorio en una carpeta
2) navegar hasta la carpeta donde se clono el repositorio
3) ejecutar terminal
4) ingresar comando:
   "npm install"
5) luego ejecutar comando:
  "npm start"

Esto iniciara un servidor local
"localhost:3000"
